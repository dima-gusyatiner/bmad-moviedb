# MovieDB Browser

A movie browsing app powered by [TMDB](https://www.themoviedb.org/). Browse popular and trending films, search by title, and view detailed movie info including cast, ratings, and backdrops.

Built with React 19, TypeScript, Vite, and Tailwind CSS 4.

## Quick Start

```bash
npm install
cp .env.example .env   # add your TMDB API key
npm run dev             # http://localhost:5173
```

Get a free API key at [themoviedb.org/settings/api](https://www.themoviedb.org/settings/api).

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Type-check and build for production |
| `npm run preview` | Preview production build |
